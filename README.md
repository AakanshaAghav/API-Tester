## API Tester

API Tester is a full-stack web application inspired by Postman that enables users to test, manage, and analyze APIs through an intuitive interface. The application supports sending HTTP requests, viewing responses, managing request history, and organizing APIs into reusable collections.

The project consists of a React/Next.js frontend for user interaction and a Node.js/Express backend for request handling and data management. Supabase PostgreSQL is used for storing request history and API collections, providing a seamless and efficient API testing experience.

## Features

* Send **GET**, **POST**, **PUT**, and **DELETE** requests.
* View response **status codes**, **headers**, and **response bodies**.
* Save and manage API request history.
* Create and organize reusable API collections.
* Full-stack architecture with separate frontend and backend modules.
* Persistent data storage using Supabase PostgreSQL.
* Responsive and user-friendly interface.
